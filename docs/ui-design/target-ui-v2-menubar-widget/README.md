# OpenStaff UI 设计图 v2（菜单栏 + 前台部件）

本设计集只用于 UI 方案确认，不涉及代码实现。

## 文件清单

- `01-menubar-dropdown.svg`
  - 菜单栏小图标及下拉菜单设计
  - 条目：打开控制台、显示/隐藏部件、切换精简/详细模式、刷新任务、紧急停止、退出程序
- `02-widget-compact-mode.svg`
  - 前台部件精简模式（球/椭圆）
  - 显示当前任务 + 下一步任务（缩略）
- `03-widget-detailed-timeline.svg`
  - 前台部件详细模式（时间轴）
  - 显示任务执行顺序、任务名称、时间、一级任务、二级任务
- `04-widget-interaction-flow.svg`
  - 交互流与状态切换图
  - 说明菜单栏切换与点击部件切换两条路径

## 需求对齐

- 菜单栏图标：已覆盖。
- 菜单栏条目中切换模式：已覆盖。
- 精简模式：球/椭圆 + 当前/下一步缩略：已覆盖。
- 详细模式：时间轴 + 执行顺序 + 一级/二级任务 + 时间：已覆盖。
- 点击球展开时间轴：已覆盖。

## 视觉方向

- 风格：轻控制台 + 玻璃浮层 + 高对比任务状态色。
- 颜色：蓝绿为主，橙色用于辅助模式，红色用于安全动作。
- 字体建议：Avenir Next / PingFang SC（macOS 兼容）。
