# OpenStaff UI 设计图 v4（Refined Minimal）

本版在 v3 基础上细化：

- 字体大小层级（标题 / 一级任务 / 二级任务 / 时间 / 辅助说明）
- 时间轴节点间距（一级节点、二级节点、组间距）
- 任务文字截断规则（单行省略、多行限制、优先保留字段）
- 精简模式从“任务球”改为“透明方框”，降低干扰

## 文件清单

- `01-menubar-native-refined.svg`
  - 菜单栏原生透明风格细化（系统字重、行距、危险动作对比）
- `02-widget-compact-box-minimal.svg`
  - 精简模式方框版（极简透明，不打扰）
- `03-timeline-embedded-refined.svg`
  - 无背景时间轴细化（节点间距、字体层级、截断示例）
- `04-typography-spacing-truncation-spec.md`
  - 可落地的尺寸与规则规范（便于后续实现）

## 关键改动

1. 精简模式：球体改为圆角方框，去除高亮渐变，透明度降低。
2. 时间轴：统一垂直节奏，一级节点和二级节点间距明确。
3. 文案：定义截断优先级，保证先看见“模式 + taskId + 状态”。
