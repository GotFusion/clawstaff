# OpenStaff v4 视觉细化规范

## 1) 字体大小层级

- 菜单栏下拉标题：`13px / weight 500`
- 菜单栏条目：`16px / weight 500`
- 菜单栏危险条目：`16px / weight 600 / red-700`
- 精简方框主行（当前任务）：`14px / weight 600`
- 精简方框次行（下一步）：`13px / weight 500`
- 时间轴一级任务标题：`16px / weight 600`
- 时间轴一级任务元信息（模式/时间）：`12px / weight 500 / mono`
- 时间轴二级任务标题：`13px / weight 500`
- 时间轴二级任务说明：`12px / weight 500`
- 规则说明文字：`12px / weight 500`

## 2) 时间轴节点间距

- 一级节点纵向最小间距：`184px`
- 一级节点标题到元信息：`22px`
- 一级节点元信息到摘要：`20px`
- 一级节点到第一条二级任务：`34px`
- 二级任务条目间距：`46px`
- 组间附加留白（一级任务结束到下一个一级任务）：`38px`
- 轨道左侧安全边距：`64px`
- 文本起始列（相对轨道）：`+28px`

## 3) 文字截断规则

### 3.1 精简方框
- 当前任务：单行，最多 `22` 个中文字符（或等宽英文 36 字符）
- 下一步任务：单行，最多 `26` 个中文字符（或等宽英文 42 字符）
- 超出统一尾部省略号 `...`

### 3.2 详细时间轴
- 一级任务标题：单行，优先保留 `模式 + taskId`，超出截断任务描述
- 一级任务摘要：单行，最长 `44` 字符
- 二级任务标题：单行，最长 `42` 字符
- 二级任务说明：单行，最长 `52` 字符

### 3.3 优先保留顺序
1. `模式`（教学/辅助/学生）
2. `taskId`
3. `状态码`（例如 `ASSIST_ACTION_EXECUTED`）
4. 其余描述

## 4) 透明度与干扰控制

- 精简方框底色：`white @ 16% ~ 20%`
- 边框：`white @ 38%`
- 字体主色：`white @ 92%`
- 字体次色：`white @ 72%`
- 一级节点（模式色）：`fill @ 42% ~ 46%`，`stroke @ 58% ~ 62%`
- 二级节点（模式色）：`fill @ 30% ~ 34%`，`stroke @ 45% ~ 48%`
- 轨道线：`blue-gray @ 70% ~ 90%`，避免高对比抢视觉
- 禁用高饱和呼吸动画，只保留静态弱阴影

## 5) 可点击反馈（轻量）

- hover：背景亮度 `+6%`
- pressed：背景亮度 `-8%`
- 动效时长：`120ms`
